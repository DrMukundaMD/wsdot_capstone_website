Safety Visualization System

What Does it Do?

What Does It Do?

The Visualization Systems allows for users to visualize current crash data on a specific portion of road. With that they are able to better understand the road and how different road attributes contribute to crashes. This system allows State Transportation employees to make better decisions about future construction.

Features

Features

Select beginning and end arms to zoom in on a smaller portion of the route. A plane graph comparison of attributes is displayed upon selection.

Mean, standard deviation, median, max, and min are generated for attributes common between beginning and end ARMs and displayed in the 'Attribute Stats' table. Hover over the attribute acronyms for a full-length description.



Experiment with new values for a given attribute and calculate the expected crash count based on that value. Check those attributes you wish to include in your calculations and unselect those that you don't.



Plot attributes against eachother by first selecting beginning and end arms then choosing desired attributes from the list of shared attributes.
